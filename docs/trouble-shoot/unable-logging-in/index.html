ログインできない

ログインできないときは、様々な理由が考えられます。

  • ログインに何度も失敗したとき
  • メールアドレス認証が住んでいないとき
  • アカウントがまだ有効になっていないとき

以下で、状況別に解説していきます。

ログインボタンを押しても、404ページが表示される#

404ページとは、該当のページが見つからなかったことを示すエラーのページです。

この現象は、ログインに何度も失敗した場合に起こります。 以下いずれかの対応をお取りください。
  • 時間をおいて(約1時間)再度試す
  • パスワードを忘れた場合は、むやみに試さずパスワードをリセットする
  • ほかの端末から試行する
  • お急ぎの場合は、AJR ライター支援に問い合わせる

ログイン時にエラーが表示される#

ERROR: User has not been activated#

このエラーは、ユーザーが有効になっていないためログインできないことを示しています。
大変申し訳ございませんが、この場合は運営にお問い合わせいただきますようお願いいたします。

ERROR: ただいまログインを制限しております#

「ログインボタンを押しても、404ページが表示される」を参照のこと

Error 403・Error 503・Error 523#

この場合、一時的にサーバーへの接続が不安定である可能性があります。5～10分程度時間をおいてからお試しください。

その他の問題#

AJR ライター支援にお問い合わせください。
