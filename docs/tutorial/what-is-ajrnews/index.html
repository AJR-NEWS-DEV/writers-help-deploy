AJR-NEWS.comとは

AJR-NEWS.com って？#

AJR-NEWS.comでの記事執筆に興味をお持ちいただき、ありがとうございます。

AJR-NEWS.comは、登録したユーザーが記事を執筆する、全く新しい鉄道情報サイトです。
ユーザー登録を行うことによって、あなたもライターとして活動いただけます。

当サイトを利用するメリット#

皆さんの経験・知識を簡単に、より多くの人へ共有できる#

あなたがこれまでにおこな行った鉄道の旅や、自分しか知らないようなお役立ち情報。
AJR-NEWSでは、それらの情報を気軽に、広く共有できます。
あなたの体験を、世界に発信してみませんか？

各種設定を自分でする必要がない#

自前でブログを始めるのはとても大変で、手間のかかることです。しかし、当サイトはサイトのデザインを作ったりする必要は一切ありません。

加えて、当サイトでは鉄道コムやGoogleニュースなどの外部プラットフォームにも記事の提供を行っていますので、より多くの人に見てもらいやすくなっています。

記事を書くには、アカウントを作成するだけでOK。簡単でしょう？
