アカウント有効化のメール#

お知らせ

この情報は、ライター登録フォームが未完成の間や問題発生時に、手動承認制での登録となっている際にのみ必要です。

アカウントが有効化されると、登録されたメールアドレス宛にアカウント有効化のお知らせが届きます。
文面は以下のようになっています。

この度は、 AJR-NEWS.com にご登録いただきありがとうございます。
審査の結果、登録が承認されましたので、会員情報と共にお届けいたします。
ユーザーID: 登録フォームで登録したIDが記載されます。
パスワード: (登録フォームで記入したパスワードをお使いください)
さっそく、以下よりログインして、プロフィール画像などの残りの設定を行ってください。
https://www.ajr-news.com/wp-login.php
敬具
AJR-News.com管理人
----------------------------------
このメールは AJR-NEWS.com より自動で送信しています。
このアドレスに返信しても運営には届きませんのでご注意ください。

このメールが届きましたら、アカウントが有効化された証です。さっそくログインしましょう！

ログインする#

メールに記載されているログインリンクか、こちらからログインできます。

トリビア

なお、今後ログインする際は、メニュー右上のボタンをご利用いただけます。

２つの管理画面#

ログインしたら、以下のような管理画面が表示されます。
この画面のことを、新管理画面といいます。

新管理画面では、AJRの運営からのお知らせや、プロフィールの設定、記事の新規投稿ができます。
また、新管理画面が工事中の間は、Wordpress標準の管理画面（以下、wp-adminという）も使用可能です。

なお、２つの管理画面は行き来できます。
wp-admin側のナビゲーションバーに「AJR新管理画面」というボタンを設けているほか、新管理画面には左下に「管理画面はこちら」というリンクを設けています。ぜひご活用ください。

プロフィール画像の設定#

プロフィール画像は、wp-adminから行えます。以下いずれかの方法で設定画面へアクセスできます。

  • あなたが、今新管理画面にいる場合
    1. 「プロフィール設定」を選択します。
    2. 現在、この画面でプロフィール画像の設定を行うことはできませんの下にあるリンクをクリック
  • あなたが、今 wp-admin にいる場合
    1. 「プロフィール」→「プロフィール」の順に選択
    2. 「プロフィール画像」の項目までスクロール

最終的に、このような画面にたどり着きます。

続いて、

  1. 「選択」のボタンを押す
  2. 「画像を選択してください」のダイアログが出るので、ファイルをアップロードして指示に従う
    重要

    著作権侵害にあたるアイコン（アニメキャラクター）などを設定した場合、著作権侵害により罰せられます。
    このようなアイコンが設定されている場合、アイコンの変更要請をさせていただいていますが、応答がない場合は予告なく画像をデフォルトに戻すことがありますのでご了承ください。

  3. 下までスクロールし、「プロフィールを更新」ボタンを押す
注意!!

必ず「プロフィールを更新」ボタンを押してください。

その他のプロフィール設定#

ライター登録フォームにて記入しなかった項目がありますか？あるなら今すぐ設定しましょう。

  • あなたが、今新管理画面にいる場合
    1. 「プロフィール設定」を選択します。
    2. 各種プロフィールを書き換え（記入）する
    3. 「プロフィールを更新」ボタンを押す
  • あなたが、今 wp-admin にいる場合
    1. 「プロフィール」→「プロフィール」の順に選択
    2. 各種プロフィールを書き換え（記入）する
    3. 「プロフィールを更新」ボタンを押す

これで、基本的な設定は終了です。さあ、次は記事の執筆をしてみましょう！
