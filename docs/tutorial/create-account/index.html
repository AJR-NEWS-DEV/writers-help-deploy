アカウントを作成してみよう！

それでは、実際にアカウントを作成してみましょう。

重要なお知らせ

現在、アカウント作成システムの修正中ですので、登録が手動承認制となっています。
ただし、登録用フォームは正常に機能します。
申し訳ございませんが、アカウントが有効化されましたらこちらからメールをお送りいたしますので、こまめにメールボックスを確認の上しばらくお待ちください。

アカウント作成に必要な情報#

アカウントを作成するために、いくつかの情報を準備してください。

必須情報#

これらの情報は、あなたとアカウントを紐づける重要な情報です。

  • あなたの電子メールアドレス
    （アカウントの発行時のトラブル回避のため、捨てメールアドレス禁止）
  • ニックネーム（ユーザー表示名 にて使用）
    （本名は使用しないことをお勧めします）

任意情報#

これらの情報は、各ユーザーのプロフィールページに表示されます。

  • 肩書き
  • 自己紹介文
  • Twitter ID
    （注：AJR更新速報BotにてこのIDが自動でメンションされます）
  • Instagram ID
  • YouTube チャンネルURL
  • 任意のURL

アカウント作成手順#

注意

アカウントを作成された場合、以下に同意したとみなします。
先にお読みください。

  • 利用規約(全体 / ライター追加規約)
  • プライバシーポリシー

アカウント申請画面へ#

図のトップページの□で囲まれたボタン、もしくはこちらより登録画面を表示します。

必要な情報を入力#

フォームに従って、必要事項を記入していってください。

  • ユーザーID … ログインの際に使用できます。これ以降変更できません。
  • ユーザー表示名 … 一般に表示される名前です。
  • メール … ログインの際に使用できます。当方からのお知らせはすべてこのメールアドレス宛に送信します。
  • 肩書きなど … 任意。ユーザー表示名の下に表示できます。
  • 自己紹介文 … 任意（入力推奨）。
  • ウェブサイト … 任意。各種ページのリンク集に掲載されます。
  • ウェブサイトの説明 … 任意。ウェブサイトのリンク名です。空白にすると「ウェブサイト」と表示されます。
  • Twitter ID … 任意。@マークより先のIDを入力してください。
  • Instagram ID … 任意。
  • YouTube チャンネルURL … 任意。URLを全て入力してください。
  • パスワード … 必須。安全のため、パスワードを8文字以上にすることを強く推奨します。
  • 「私はロボットではありません」 … 必須。安全担保のため、ご協力をお願いいたします。

メールアドレス認証#

すべての項目を入力したら、「登録」ボタンを押します。すると、「ライター登録フォーム」のセクションのフォームが以下の文章に変化します。
(なお、文面は少し変化する可能性があります)

登録に成功しました。記入いただいたメールアドレスあてにログインのための情報が記載されています。
迷惑メールに振り分けられていることが多いようです。届かない場合には一度確認してください。

そうしたら、フォームに入力いただいたメールアドレスにメールが送信されていますので、迷惑メールフォルダを含めてご確認ください。

当サイトからのアカウント関連のメールはuser-manage@ajr-news.comから送信されます。
メールのフィルタリング機能などを有効にされている方は、先にこのアドレスをホワイトリストに含んでください。

メールの文面は、以下のようになっています。

メールの末尾に、「メールアドレス確認用リンク」がある場合は、6時間以内に必ずクリックしてください。

リンクをクリックすると、以下のようなメッセージが表示されます（文面は変わることがあります）。

メッセージの末尾にYour account is now pending approvalと表示される場合、まだログインできません。
